emotion detector

1. Get the connected cameras, then select a camera.
2. Select if you want to write to send OSC data, write to a logfile, or both. The location of the logfile (on the Mac) is: ~Library/Logs/emotion-logger/.
3. Switch on the detector.
When a face is detected, the probabilities of the different emotions are sent via OSC / written to the log file.
When done, switch off the detector. Move or rename the logfile before the next session.
